# Login Test Cases

| Test Case ID | Title            | Steps                                                     | Expected Result                     |
| ------------ | ---------------- | --------------------------------------------------------- | ----------------------------------- |
| TC-001       | Successful Login | Enter valid username and password and click Login         | User is redirected to Products page |
| TC-002       | Invalid Username | Enter invalid username and valid password and click Login | Error message is displayed          |
| TC-003       | Invalid Password | Enter valid username and invalid password and click Login | Error message is displayed          |
| TC-004       | Empty Username   | Leave username empty and click Login                      | Error message is displayed          |
| TC-005       | Empty Password   | Leave password empty and click Login                      | Error message is displayed          |
