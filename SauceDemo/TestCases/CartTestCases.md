# Cart Test Cases

| Test Case ID | Title                           | Expected Result                              |
| ------------ | ------------------------------- | -------------------------------------------- |
| TC-CART-001  | Add one product to cart         | Product is added successfully                |
| TC-CART-002  | Add multiple products to cart   | All selected products are added successfully |
| TC-CART-003  | Remove product from cart        | Product is removed successfully              |
| TC-CART-004  | Open cart after adding products | Added products are displayed in cart         |
| TC-CART-005  | Proceed to checkout from cart   | User is redirected to checkout page          |
| TC-CART-006  | Continue shopping from cart     | User is redirected to products page          |
