# SQL Practice

## Overview

This project contains SQL exercises focused on Quality Assurance activities.

The purpose is to learn how QA professionals use SQL to validate data, verify application records, investigate defects, and support testing activities.

## Topics Covered

### Basic Queries

* SELECT
* WHERE
* ORDER BY

### Aggregate Functions

* COUNT()
* AVG()
* MIN()
* MAX()
* SUM()

### Table Relationships

* INNER JOIN
* LEFT JOIN

### Data Analysis

* GROUP BY
* HAVING
