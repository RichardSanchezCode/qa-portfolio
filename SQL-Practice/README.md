# SQL Practice

## Overview

This project contains SQL exercises focused on Quality Assurance activities.

The purpose is to learn how QA professionals use SQL to validate application data, verify database records, and support testing activities.

## Topics Covered

* SELECT Statements
* Data Filtering
* Sorting Results
* Aggregate Functions
* Data Validation

## Learning Objectives

* Retrieve data from database tables
* Filter records using conditions
* Sort query results
* Analyze data using aggregate functions
* Apply SQL skills to QA scenarios
