# Exploratory Testing Project

## Overview

This project contains exploratory testing sessions performed on a QA practice web application.

The objective was to identify defects, validate application behavior, document observations, and improve bug reporting skills through hands-on testing.

## Application Tested

* The Internet (QA Practice Application)

## Testing Type

* Exploratory Testing
* Functional Testing
* UI Testing
* Session Testing
* Authentication Testing

## Modules Explored

### Form Authentication

Activities performed:

* Successful login validation
* Invalid username validation
* Invalid password validation
* Empty credentials validation
* Session management testing
* Protected URL access testing
* Logout verification

Result: PASS

---

### Dynamic Controls

Activities performed:

* Checkbox removal and restoration
* Dynamic element validation
* Input enable/disable validation
* State persistence verification
* Repeated action testing

Result: 1 Defect Identified

---

### Dropdown

Activities performed:

* Option selection validation
* Default option behavior verification
* Page refresh behavior testing
* Keyboard and interaction testing

Result: PASS

---

### Inputs

Activities performed:

* Positive number validation
* Negative number validation
* Integer validation
* Decimal value validation
* Input control behavior testing

Result: PASS
