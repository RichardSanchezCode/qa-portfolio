# SauceDemo Test Plan

## Project Information

**Project Name:** SauceDemo
**Application Type:** E-commerce Web Application
**Testing Type:** Manual Testing

## Objective

Verify that the main functionalities of SauceDemo work correctly and meet the expected requirements.

## Scope

The following modules will be tested:

* Login
* Products
* Cart
* Checkout

## Out of Scope

* Performance Testing
* Security Testing
* Mobile Testing

## Test Environment

* Browser: Google Chrome
* Operating System: Windows

## Deliverables

* Test Cases
* Bug Reports
* Test Evidence
