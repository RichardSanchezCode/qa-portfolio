# Products Test Cases

| Test Case ID   | Title                                | Expected Result                                             |
| -------------- | ------------------------------------ | ----------------------------------------------------------- |
| TC-PRODUCT-001 | View products list after login       | Products page is displayed successfully                     |
| TC-PRODUCT-002 | Verify all products are displayed    | All available products are visible                          |
| TC-PRODUCT-003 | Add a product from Products page     | Product is added to cart successfully                       |
| TC-PRODUCT-004 | Remove a product from Products page  | Product is removed from cart successfully                   |
| TC-PRODUCT-005 | Sort products from A to Z            | Products are sorted alphabetically (A-Z)                    |
| TC-PRODUCT-006 | Sort products from Z to A            | Products are sorted alphabetically (Z-A)                    |
| TC-PRODUCT-007 | Sort products by price (Low to High) | Products are sorted by ascending price                      |
| TC-PRODUCT-008 | Sort products by price (High to Low) | Products are sorted by descending price                     |
| TC-PRODUCT-009 | Open product details                 | Product details page is displayed correctly                 |
| TC-PRODUCT-010 | Verify cart badge updates            | Cart badge displays the correct number of selected products |

## Notes

The objective of these test cases is to validate product listing, sorting functionality, product details, cart integration, and user navigation within the Products module.
