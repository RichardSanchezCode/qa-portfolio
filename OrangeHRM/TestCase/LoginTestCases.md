# Login Test Cases

| Test Case ID | Title                                  | Expected Result                 |
| ------------ | -------------------------------------- | ------------------------------- |
| TC-LOGIN-001 | Login with valid credentials           | User is redirected to Dashboard |
| TC-LOGIN-002 | Login with invalid username            | Error message is displayed      |
| TC-LOGIN-003 | Login with invalid password            | Error message is displayed      |
| TC-LOGIN-004 | Login with empty username              | Validation message is displayed |
| TC-LOGIN-005 | Login with empty password              | Validation message is displayed |
| TC-LOGIN-006 | Login with empty username and password | Validation message is displayed |
