# OrangeHRM Test Plan

## Project Information

**Project Name:** OrangeHRM

**Application Type:** Human Resource Management System (HRMS)

**Testing Type:** Manual Testing

## Objective

Verify that the core functionalities of OrangeHRM operate correctly and meet expected business requirements.

## Scope

The following modules will be tested:

* Login
* Dashboard
* Admin
* PIM (Personal Information Management)
* Leave
* Recruitment

## Out of Scope

* Performance Testing
* Security Testing
* Mobile Testing

## Test Environment

* Browser: Google Chrome
* Operating System: Windows

## Deliverables

* Test Cases
* Test Execution Results
* Bug Reports
* Test Evidence
